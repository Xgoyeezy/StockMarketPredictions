# Contributing

Thanks for taking a look at Quant Evidence OS / StockMarketPredictions.

## How to suggest changes

Use one of these paths:

- Open an issue for bugs, questions, missing documentation, or feature suggestions.
- Start a discussion for broader architecture, research workflow, or positioning feedback.
- Submit a pull request if you want to propose a concrete code or documentation change.

Direct changes to `main` are restricted. Pull requests require review before merge.

## Boundaries

Do not propose changes that:

- Enable autonomous live-money trading.
- Give AI authority to place orders.
- Bypass risk gates.
- Auto-clear kill switches.
- Loosen broker routes automatically.
- Let analytics change ranking weights automatically.
- Merge simulation evidence into market-observed evidence.
- Add broker account IDs, raw broker records, credentials, secrets, or raw local paths.

## Suggested contribution areas

- Documentation clarity.
- Test coverage.
- UI usability.
- Evidence review workflows.
- Forecast validation reporting.
- Benchmark and walk-forward proof surfaces.
- Safer empty states and operator summaries.
