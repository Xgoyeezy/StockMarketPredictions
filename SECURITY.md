# Security

Do not post secrets, broker credentials, account IDs, raw broker records, raw logs, or private local file paths in issues, discussions, or pull requests.

If you notice a security problem, open a GitHub issue with a high-level description only. Do not include exploit details or sensitive data publicly.

This project is research-first and paper-first. It should not be treated as a broker control replacement, compliance-approved system, or autonomous money manager.
